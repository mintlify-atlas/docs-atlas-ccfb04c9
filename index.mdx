<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-semibold dark:text-gray-100 text-gray-900 mb-4">Quick start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get CooperBench running in minutes</p>
  
  <Steps>
    <Step title="Install CooperBench">
      Install the package via pip:
      
      ```bash
      pip install cooperbench
      ```
      
      For development, clone the repository:
      
      ```bash
      git clone https://github.com/cooperbench/CooperBench.git
      cd CooperBench
      pip install -e ".[dev]"
      ```
    </Step>
    
    <Step title="Set up execution backend">
      Choose an execution backend and configure it:
      
      **Option 1: Modal (default)**
      ```bash
      modal setup
      ```
      
      **Option 2: GCP**
      ```bash
      pip install 'cooperbench[gcp]'
      cooperbench config gcp
      ```
      
      **Option 3: Docker (local)**
      
      No additional setup required — just have Docker running.
    </Step>
    
    <Step title="Configure Redis and LLM API keys">
      Start Redis for inter-agent communication:
      
      ```bash
      docker run -p 6379:6379 redis:7
      ```
      
      Set your LLM API keys in a `.env` file:
      
      ```bash
      ANTHROPIC_API_KEY=your_key
      OPENAI_API_KEY=your_key
      GEMINI_API_KEY=your_key
      ```
    </Step>
    
    <Step title="Run your first experiment">
      Run cooperative agents on a task:
      
      ```bash
      cooperbench run -n my-experiment -r llama_index_task -m gpt-4o
      ```
      
      Evaluate the results:
      
      ```bash
      cooperbench eval -n my-experiment
      ```
      
      <Accordion title="View example output">
        Results are saved to `logs/my-experiment/`:
        
        ```
        logs/my-experiment/llama_index_task/task1234/features_1_2/
          agent1/
            trajectory.json
            patch.diff
          agent2/
            trajectory.json
            patch.diff
          eval.json
        ```
      </Accordion>
    </Step>
  </Steps>
</div>
